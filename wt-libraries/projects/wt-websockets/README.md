# Wt Websockets
Library used to connect with socket io. It has a module that needs to be provided witin the app configuration and its variables used to configured the server.


## Build library in dev mode
To build this library in dev mode run
```bash
  ng build --watch
```

## Dependencies
### ngx socket io
```bash
   npm install ngx-socket-io socket.io-client
   npm install @types/socket.io-client
```
